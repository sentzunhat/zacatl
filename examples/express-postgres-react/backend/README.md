# Zacatl Express + PostgreSQL Backend

Minimal Express backend for the Express + PostgreSQL + React example.

## Quick Start

```bash
npm install
npm run dev
```

- API: http://localhost:8183
- Default DB URL: postgres://local:local@localhost:5432/appdb

## Environment

- PORT (default: 8183)
- DATABASE_URL (default: postgres://local:local@localhost:5432/appdb)
