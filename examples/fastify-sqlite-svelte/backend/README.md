# Zacatl Fastify + SQLite Backend

Minimal Fastify backend for the Fastify + SQLite + Svelte example.

## Quick Start

```bash
npm install
npm run dev
```

- API: http://localhost:8081
- Default DB: sqlite:database.sqlite

## Environment

- PORT (default: 8081)
- DATABASE_URL (default: sqlite:database.sqlite)
