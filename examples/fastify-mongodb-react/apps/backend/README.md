# Zacatl Fastify + MongoDB Backend

Minimal Fastify backend for the Fastify + MongoDB + React example.

## Quick Start

```bash
npm install
npm run dev
```

- API: http://localhost:8082
- Default Mongo URI: mongodb://localhost:27017/greetings-db

## Environment

- PORT (default: 8082)
- MONGO_URI (default: mongodb://localhost:27017/greetings-db)
