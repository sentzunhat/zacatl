# Zacatl Fastify + MongoDB Backend

Minimal Fastify backend for the Fastify + MongoDB + React example.

## Quick Start

```bash
npm install
npm run dev
```

- API: http://localhost:8082
- Default Mongo URI: mongodb://local:local@localhost:27017/appdb

## Environment

- PORT (default: 8082)
- MONGO_URI (default: mongodb://local:local@localhost:27017/appdb)
