# Express + Mongoose + MongoDB Example

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Access the API at [http://localhost:8084/greetings](http://localhost:8084/greetings)

## Endpoints

- `GET /greetings`: Get all greetings
- `GET /greetings/:id`: Get greeting by ID
- `POST /greetings`: Create a new greeting
- `DELETE /greetings/:id`: Delete a greeting
- `GET /greetings/random`: Get a random greeting
