# Internationalization API

Multi-language support with i18next and native Intl.

## Import

```typescript
import { createI18n } from "@sentzunhat/zacatl";
import { I18NextAdapter } from "@sentzunhat/zacatl/localization/adapters";
```

## Basic Setup

```typescript
const i18n = createI18n(
  new I18NextAdapter({
    lng: "en",
    fallbackLng: "en",
    resources: {
      en: {
        translation: {
          welcome: "Welcome",
          goodbye: "Goodbye",
        },
      },
      es: {
        translation: {
          welcome: "Bienvenido",
          goodbye: "Adiós",
        },
      },
    },
  }),
);
```

## Translate

```typescript
i18n.t("welcome"); // "Welcome"

await i18n.setLanguage("es");
i18n.t("welcome"); // "Bienvenido"
```

## With Variables

```typescript
resources: {
  en: {
    translation: {
      greeting: "Hello, {{name}}!";
    }
  }
}

i18n.t("greeting", { name: "John" }); // "Hello, John!"
```

## Load from Files

```typescript
import i18next from "i18next";
import Backend from "i18next-fs-backend";

await i18next.use(Backend).init({
  lng: "en",
  fallbackLng: "en",
  backend: {
    loadPath: "./locales/{{lng}}/{{ns}}.json",
  },
});

const i18n = createI18n(new I18NextAdapter(i18next));
```

## File Structure

```
locales/
├── en/
│   └── translation.json
├── es/
│   └── translation.json
└── fr/
    └── translation.json
```
