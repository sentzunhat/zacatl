# Zacatl Context Pack

This directory provides a compact context pack for AI agents and contributors.
It follows a What / How / Why split and adds focused API, security, and
performance references.

## Files

- `context.yml` - What the repository is and how modules are organized
- `guidelines.yml` - How to implement changes consistently
- `patterns.yml` - Why key architectural decisions exist
- `api.yml` - Service and integration API guidance
- `security.yml` - Dependency, auth, and runtime security guidance
- `performance.yml` - Build/runtime and docs performance guidance

## Recommended Read Order

1. `context.yml`
2. `guidelines.yml`
3. `patterns.yml`
4. `api.yml`, `security.yml`, `performance.yml` as needed

## Scope

This context pack is documentation support only. For authoritative standards,
keep `docs/guidelines/` as source of truth.
